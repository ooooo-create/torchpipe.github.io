针对配置文件，我们提供了简单的网页可视化功能。

## 环境准备
```bash
apt-get update
apt install graphviz
pip install pydot gradio
```

## 使用方法 {#parameter}

 `torchpipe.utils.vis [-h] [--port PORT] [--save] toml`

:::tip 参数
- **--save** - 是否将图保存为svg图片。图片将保存在当前目录下，与toml文件（后缀不同）
:::


## 示例
```bash
python -m torchpipe.utils.vis your.toml # --port 2211
```
