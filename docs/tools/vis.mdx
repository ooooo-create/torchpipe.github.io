(From v0.4.0)
We provide a simple web-based visualization feature for configuration files.
## Environment Setup

```bash
pip install  gradio
```

## Usage {#parameter}


```python
import torchpipe as tp

a=tp.parse_toml("examples/ppocr/ocr.toml")

tp.utils.Visual(a).launch()
```
