(From v0.4.0)
We provide a simple web-based visualization feature for configuration files.
## Environment Setup

```bash
pip install  gradio pyvis
```

## Usage {#parameter}


```python
import torchpipe as tp


tp.utils.Visual("examples/ppocr/ocr.toml").launch()
```
